# Product Backlog: Batalha Naval Online (Papergames.io)

Este backlog descreve as funcionalidades principais da interface web do jogo Batalha Naval, estruturadas sob a forma de Scrum User Stories para uma equipa de 4 pessoas.

## Épico 1: Conta e Menu Principal (Ideal para o Elemento 1)
- **US01:** Como utilizador, quero poder introduzir um "Nickname" na página inicial para me identificar perante os adversários.
- **US02:** Como utilizador, quero ter a opção de clicar em "Login / Sign Up" para poder guardar o meu progresso e avatar.
- **US03:** Como utilizador, quero ter um botão "Leaderboard" para poder visualizar o ranking dos melhores jogadores.
- **US04:** Como utilizador, quero aceder à secção "How to Play" (regras) para ler as instruções da Batalha Naval antes de jogar.

## Épico 2: Configuração de Partida (Ideal para o Elemento 2)
- **US05:** Como jogador, quero ter um botão "Play with a Friend" para gerar um link de convite único que posso partilhar.
- **US06:** Como iniciante, quero ter a opção "Play vs Bot" para iniciar imediatamente uma partida contra o computador.
- **US07:** Como utilizador, quero poder abrir a janela de "Settings" (roda dentada) para ajustar o volume do som do jogo.
- **US08:** Como utilizador, na janela de "Settings", quero poder alterar o idioma da interface.

## Épico 3: Posicionamento da Frota (Ideal para o Elemento 3)
- **US09:** Como jogador, na fase de preparação, quero clicar no botão "Randomize" para que o sistema posicione os meus navios aleatoriamente.
- **US10:** Como jogador tático, quero poder arrastar (drag and drop) os meus navios da doca para o tabuleiro para escolher a sua posição.
- **US11:** Como jogador, quero poder clicar num navio enquanto o arrasto para o rodar entre a posição horizontal e vertical.
- **US12:** Como jogador, após colocar toda a minha frota validamente, quero que o botão "Ready" fique ativado para sinalizar que estou pronto.

## Épico 4: Gameplay e Fim de Jogo (Ideal para o Elemento 4)
- **US13:** Como jogador, durante o meu turno, quero poder clicar numa quadrícula do tabuleiro do adversário para disparar um tiro.
- **US14:** Como jogador em partida, quero ver um temporizador (timer) no ecrã para saber quanto tempo tenho para fazer a minha jogada antes de perder a vez.
- **US15:** Como jogador que precisa de sair, quero ter um botão de "Surrender/Leave" para abandonar a partida atual.
- **US16:** Como jogador no final de uma partida, quero ver um botão de "Rematch" (Desforra) para convidar o mesmo adversário para um novo jogo.
